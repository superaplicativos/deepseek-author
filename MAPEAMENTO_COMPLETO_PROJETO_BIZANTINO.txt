1. VISÃO GERAL DO PROJETO
================================================================================

PROPÓSITO E CONTEXTO
--------------------
Este é um projeto de PUBLICAÇÃO AUTOMATIZADA DE LIVROS INFANTIS usando IA
(Claude AI), operando como um "clone brasileiro" da startup israelense Spines
(que levantou $16M).

O sistema:
• Produz livros de 35-50 mil palavras em ~3 horas
• Custo de $10-25 por livro (vs $1.200-$5.000 da Spines)
• Automação de 95% do processo
• Publicação simultânea em 4 idiomas (PT, EN, DE, ES)
• Distribuição em 10 países via Amazon KDP

PERFORMANCE COMERCIAL
----------------------
• 331 vendas em 10 países
• 24.823 páginas KENP (Kindle Unlimited) lidas
• $377 USD + R$434 BRL em receita
• $0 gastos em marketing
• Vendas triplicando mensalmente

C. SCRIPTS/ - Automação do Pipeline de Produção
------------------------------------------------
Organização: 6 subpastas por função, 25+ scripts Python.

1-MANUSCRITO/ (3 scripts)
  • create_docx.py - Converte manuscrito.txt do Livro 7 para DOCX
  • create_docx_egito.py - Converte manuscrito do Livro 8 para DOCX
  • generate_prompts.py - Gera prompts de IA para ilustrações

2-TRADUCAO/ (7 scripts) - SISTEMA CRÍTICO
  • tradutor_docx_GRATUITO.py (366 linhas) - SCRIPT PRINCIPAL
    - 100% gratuito usando Google Translate via web
    - Preserva TODA formatação e imagens
    - Suporta: en, es, de, fr, it, nl, ru, zh-cn, ja, ko
    - Usa deep-translator (sem API keys necessárias)

  • tradutor_odt_GRATUITO.py - Versão para ODT
  • tradutor_odt_direto_GRATUITO.py - Tradução direta ODT
  • qa_traducao_docx.py - QA automático pós-tradução
  • qa_fix_punctuation_docx.py - Correção de pontuação
  • qa_pontuacao_leve.py - Correção leve
  • verificar_traducao.py - Verificação completa

  Workflow típico:
    # Traduzir para Inglês
    python tradutor_docx_GRATUITO.py livro.docx en

    # QA automático
    python qa_traducao_docx.py livro-EN-GRATUITO.docx

3-FORMATACAO-KDP/ (5 scripts)
  • conversor_amazon_kdp.py - Converte DOCX para formato KDP
  • conversor_epub_kdp_DEFINITIVO.py - Conversor EPUB principal
  • conversor_epub_kdp_espanhol.py - Específico para espanhol
  • ajuste_kdp_6x9.py - Ajusta para 6x9 polegadas (padrão Amazon)
  • otimizador_espacamento_kindle.py - Otimiza para Kindle

4-ANALISE/ (5 scripts)
  • analisar_docx.py - Análise geral de DOCX
  • analisar_livro3.py - Análise específica Livro 3
  • verificar_traducoes_livro3.py - Verificação traduções
  • inspecionar_imagens.py - Inspeção de imagens em DOCX
  • corrigir_imagens_docx.py - Correção de problemas com imagens

5-MARKETING-RESEARCH/ (6 scripts)
  • amazon_keyword_mapper.py - Mapeia keywords da Amazon
  • amazon_top_terms_prefix.py - Analisa termos top com prefixos
  • global_top_1000_keywords.py - Top 1000 keywords globais
  • kdp_growth_intelligence.py - Inteligência de crescimento KDP
  • generate_kdp_html_report.py - Relatórios HTML de análise
